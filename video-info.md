# Video Signal Placeholder

Since ffmpeg is not available in this environment, the system expects a video file named `signal.mp4` to be placed in the root directory for the video caption demonstration.

The caption system includes:
- Time-synchronized captions that appear during video playback
- Signal data loading with frequency and temporal information
- Real-time caption display overlay

Sample captions are pre-loaded with mystical Wraithborne-themed messages that sync with video timestamps.
